# SkillSnap — AI Career Intelligence Platform

SkillSnap is an AI-powered platform that helps students and professionals transform resumes into actionable career insights.

## Features

* Resume Analysis with AI
* ATS Compatibility Score
* Recruiter Readiness Score
* Portfolio Generator
* AI Interview Coach
* GitHub Profile Analysis
* LinkedIn Profile Evaluation
* Weekly Skill Roadmap

## Tech Stack

Frontend

* React
* Vite
* TypeScript

Backend

* FastAPI
* Python

Database

* PostgreSQL (Neon)

AI

* Gemini API

Deployment

* Vercel
* Render
